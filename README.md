# 📻 SolarPunk Radio
## Community radio. Live stream. Programmed blocks. Contributor model. All shows archived to IPFS permanently. Free.

---

> Radio was the first internet — a signal broadcast freely into the air that anyone with a receiver could catch.
> We're building the next one. Digital. Decentralized. Permanently archived. Run by the community.

---

## The Architecture

```
CONTENT SOURCES (anyone can contribute)
    │
    ▼
ICECAST SERVER (open source, self-hosted or free-tier VPS)
    │
    ├──▶ Live stream: radio.solarpunk.earth (coming)
    ├──▶ On-demand: IPFS archives of every show
    └──▶ Embed widget: drop the player into any page
    
PROGRAMMING BLOCKS
    ├── Music: Creative Commons + artist-submitted tracks
    ├── News: curated from free press sources
    ├── Knowledge: readings from mycelium-knowledge docs
    ├── Community calls: recorded mutual aid coordination
    └── Open mic: submitted community audio
```

---

## How to Host

**Free option:** Icecast on a $5/month VPS (Hetzner, Vultr, Linode). One-time setup. Runs forever.

**Free-er option:** Use a free Icecast hosting service while traffic is small. Upgrade when the community grows.

**The mycelium way:** Multiple nodes. Each community runs its own stream. Federated schedule means content flows between stations.

---

## How to Contribute Audio

1. Record your thing. Any format. Any length.
2. Open a GitHub Issue: "SUBMISSION — [title] — [category]"
3. Attach the file or link to it
4. Community votes with reactions
5. Approved submissions get programmed and archived

---

## What Gets Played

Music, conversation, readings, news, tutorials, community calls, art, poetry, silence. The community decides what goes on. The only filter: AGPL-3.0 + Ethical Use. No content promoting violence, surveillance, or hate.

---

## The Archive

Every show gets uploaded to IPFS the moment it airs. Permanent. Content-addressed. Nobody can delete it. The archive is the permanent record of a community's voice.
