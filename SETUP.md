# SolarPunk Radio — Setup Guide

## What This Is

A 24/7 autonomous internet radio stream: lo-fi music + spoken word from the SolarPunk system + community voices, running on free or near-free infrastructure.

## Infrastructure (Zero-Cost Path)

### Option A: Completely Free (limited hours)
- **Mixcloud:** 1-hour weekly show, free upload, no stream — good for recorded shows
- **Spreaker:** Free podcast hosting, RSS, embeddable player
- **Archive.org:** Unlimited audio uploads, permanent, free — use as your CDN

### Option B: ~$5-10/month (true 24/7 stream)
- **Icecast2** on a $4/month VPS (Hetzner, Netcup, DigitalOcean)
- **Liquidsoap** for intelligent playlist generation with scripted interludes
- **Icecast** for the public stream endpoint
- **GitHub Actions** to push new content and track playlists

### Option C: Restream approach (free tier)
- **Radio.co** free trial → Mixcloud Live → YouTube Live simultaneously
- Record, then post to Archive.org for permanent free hosting

## Content Sources

### What we have right now:
- Royalty-free lo-fi music (CC0 / Creative Commons)
- Meeko's flower 3D shorts audio tracks
- SolarPunk Remedies audio guides (to be generated)

- Daily "what this is" opening monologue (generated by AI, spoken via TTS)
- Weekly grant announcements
- SolarPunk knowledge spots (rights info, free tools)

## To Start (No Money)

1. Upload tracks to archive.org (free account)
2. Create a Spreaker podcast (free)
3. Use playlist_generator.py to build show files
4. Upload weekly shows to Spreaker + Archive.org
5. Embed the player anywhere with one line of HTML

## To Scale ($5/month)

1. Spin up a Hetzner CX11 VPS (~€3.29/month)
2. Install Icecast2 + Liquidsoap
3. Set up GitHub Actions to push new content nightly
4. Point a subdomain at it
5. True 24/7 stream
